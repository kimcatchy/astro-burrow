# Project Overview

Burrow is a personal blog project focused on clean design, modularity, and seamless content management. It leverages the latest web technologies to provide a fast, accessible, and easily maintainable platform.

## Key Features

- **Framework**: Built with Astro 5.x for lightning-fast performance.
- **Styling**: Fully responsive design using Tailwind CSS v4 and custom dark/yellow theme.
- **UI Components**: High-quality UI elements provided by shadcn/ui.
- **CMS**: Integrated with Notion API for effortless writing and publishing.
- **Automation**: GitHub Actions for automated builds and updates.

## Future Roadmap

- Deep integration with Notion Database.
- Automated thumbnail generation.
- Search functionality using Pagefind.
